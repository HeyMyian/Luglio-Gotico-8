# Luglio Gotico 8

A proportional pixel font for Wulfila&rsquo;s **Gothic alphabet** which was used to write the extinct East Germanic Gothic language in the Codex Argenteus and other works.

## Installation

1. Download the [luglio-gotico-8.ttf](https://github.com/HeyMyian/Luglio-Gotico-8/raw/refs/heads/main/luglio-gotico-8.ttf) file.
2. Open the file on your computer. This should automatically open the font preview with an &ldquo;install&rdquo; button.

## Usage

### Font size

The correct font size for this pixel font is **8** (or multiplies of 8). The base glyph height is 11px with 3px ascents and 6px descents (total height of 20px).

### Reference

Reference the Wikipedia entry for the [Gothic alphabet](https://en.wikipedia.org/wiki/Gothic_alphabet).

A few things to consider:

- The `𐍁` symbol for the numeral &ldquo;90&rdquo; can be typed with `c`.
- The symbol `𐍊` for the numeral &ldquo;900&rdquo; can be typed with `0` (zero).
- Uppercase and lowercase letters are identical.
- Diacritics on the vowels `áāēíōū` are identical to the base vowels. This is for ease of copy-pasting text that contains diacritics. Exception is `ï` (i with trema), which has its own glyph.
- Punctuation `.,;?!` is all rendered as middot `·`.
- Currently, there are the following additional symbols which do not exist in the source material: `()[]`, as well as `dash`, `en-dash`, and `em-dash`.
- The glyph for the `ƕ` ligature is not yet implemented. It can be typed with this symbol or with `v`.

## Modification

1. Download the `png` and `json` files in the source folder.
2. Modify the `luglio-gotico-8.png` file in a pixel editing software of your choice. Add or modify glyphs as you see fit and make sure you save the file at 100% scale (1px ≙ 1px).
3. Open the [Pixel Font Converter](https://yal.cc/tools/pixel-font) in your browser.
4. Click on **Menu** > **Import settings** and select the `luglio-gotico-8.json` file.
5. Click on **Pick image** and select your modified `luglio-gotico-8.png` file.
6. In the **Input** tab, modify the glyphs that are present according to the layout of the `png` file.
7. In the **Meta** tab, change the information there as you see fit.
8. Click on **Save TTF** to download the new font file.
